# BurgerTokenJumpStartSplit
This Phyton script splits the output of the Burgertoken.com JumpStart Theme Card Builder into two separate pictures FRONT (left) and BACK (right)

Burgertoken Theme Card Builder access via:
https://burgertokens.com/pages/jumpstart-theme-card-builder

## How to use
Ensure you  have Phyton 3 installed
Theme card output should end with *-theme.png

##execute
split_crop_no_white_border --in "Input Folder Name" --out "Output Folder Name"
